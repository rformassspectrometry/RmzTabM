# Sort rows in a MTD matrix to match the expected order

Helper function to sort a mzTab-M *MTD* `matrix`, such as generated by
[`mtd_skeleton()`](https://rformassspectrometry.github.io/RmzTabM/reference/mtd_skeleton.md),
into the correct order of the metadata fields.

## Usage

``` r
mtd_sort(x)
```

## Arguments

- x:

  two-column matrix with the first column containing the metadata field
  names.

## Value

input parameter `x` sorted into the correct order.

## See also

[MTD-export](https://rformassspectrometry.github.io/RmzTabM/reference/MTD-export.md)
for other functions defining metadata information
